# UEOT-Complex-systems-Lab
An interactive agent-based simulation lab for Universal Evolutionary Object Theory (UEOT), exploring the emergence of order, life, and systemic collapse via topological dynamics and the $\Pi/\Phi$ variational principle. 通用演化客体理论 (UEOT) 的交互式仿真实验室，通过拓扑动力学和 $\Pi/\Phi$ 变分原理探索秩序、生命及系统性崩塌的涌现机制。
